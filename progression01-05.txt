
week1  เรียน array (push,popfrom,of) วนลูปแสดงข้อมูล จัดการ string ใช้ destructure จาก array 
เอาเนื้อหาที่อาจารย์สอน ไปให้ chatgpt สร้างโจทย์ พอเขียนเสร็จก็ใช้ chat ช่วยตรวจฃ

week2 เรื่องเปรียบเทียบ object ตรวจสอบว่า object สองตัว มีkey กับ value เหมือนกันมั้ย หัดใช้ Object.keys() 
มานับจำนวน key แล้ววนลูปเช็คค่าทีละตัว ถ้าทุกค่าเหมือนกันจะคืนค่า true ถ้ามีต่างกันจะคืนค่า false แล้วก็ลองทำความเข้าใจ code อาจารย์
หาโจทย์เพิ่ม จาก chatgpt หาคำอธิบายเพิ่ม

week3 เรียนเรื่อง Function ของ JavaScript ตั้งแต่พื้นฐาน syntax, การใช้ return, arrow function, function as object, 
การคิดแบบbasic programing หาโจทย์จาก chat ทำเพิ่ม เอาโจทย์ในห้อง มาทวนใหม่

week4 เรียนเรื่อง Array Methods  , Callback Function และ Higher-Order Function , 
เมธอด push(), pop(), shift(), unshift() , เทคนิคการ debug เวลาเขียนโปรแกรม
